Íconos para apps Electron y macOS

Electron

Las apps Electron suelen usar PNGs en tamaños 16–256. Usa IcoSmith para exportar todos los tamaños y configúralos en tu build (electron-builder).

  • Windows: provee icon.ico (multi-tamaño) y PNGs para instaladores.
  • Linux: PNGs por tamaño en resources/.

macOS (.icns)

IcoSmith exporta PNGs; para generar .icns, usa iconutil o herramientas de build (electron-builder convierte automáticamente).

  • Incluye PNGs 16, 32, 64, 128, 256, 512 y 1024 para mejor resultado.
  • Mantén márgenes y evita exceso de detalles pequeños.
